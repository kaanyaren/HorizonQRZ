# HorizonQRZ

HorizonQRZ is a modern, cross-platform (iOS & Android) mobile application designed for radio amateurs to manage their QRZ.com Logbook seamlessly.

## Key Features

- **Real-time Logging:** Log your QSOs instantly with official QRZ.com Logbook API integration.
- **Offline Mode:** Log your contacts in the field (POTA/SOTA) and sync them automatically when you're back online.
- **Callsign Lookup:** View bios, photos, and details of your contacts via the QRZ XML Callbook API.
- **Award Tracking:** Monitor your progress for DXCC, WAS, and other QRZ.com awards directly in the app.
- **Logbook Browsing:** Search and filter your past contacts with a locally cached database for speed.

## Technical Foundation

- **Framework:** Flutter
- **State Management:** Riverpod
- **Local Database:** Drift (SQLite)
- **API Integration:** Official QRZ.com XML and REST APIs.
